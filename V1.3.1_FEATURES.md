# v1.3.1 新功能说明 - 系统进程视觉区分

**更新日期**: 2026-07-08  
**版本**: v1.3.1

---

## 🎨 新增功能

### 1. 视觉区分系统进程和用户进程

**系统进程**：
- 🎨 浅灰色背景 (#E0E0E0)
- 🎨 灰色文字 (#666666)
- 🔒 状态栏标记"受保护"

**用户进程**：
- 🎨 白色背景（默认）
- 🎨 黑色文字（默认）
- ✓ 可自由操作

**效果对比**：
```
┌─────────────────────────────────────────────┐
│ PID  | 进程名称    | 类型    | 已提交 | 状态   │
├─────────────────────────────────────────────┤
│ 1234 | chrome.exe | 浏览器  | 2.5 GB | 建议   │ ← 白色背景
│ 5678 | code.exe   | 开发    | 1.8 GB | 建议   │ ← 白色背景
│ 9012 | notepad    | 用户    | 10 MB  | 正常   │ ← 白色背景
├─────────────────────────────────────────────┤
│ 100  | explorer   | 系统    | 150 MB | 受保护 │ ← 灰色背景
│ 200  | dwm        | 系统    | 80 MB  | 受保护 │ ← 灰色背景
│ 300  | svchost    | 系统    | 50 MB  | 受保护 │ ← 灰色背景
└─────────────────────────────────────────────┘
```

---

### 2. 过滤功能：隐藏系统进程

**位置**：顶部工具栏  
**控件**：复选框"隐藏系统进程"

**功能**：
- ☐ **未勾选**（默认）：显示所有进程
- ☑ **已勾选**：只显示用户程序，隐藏系统进程

**示例**：
```
未勾选时：
✓ 已加载 235 个进程（总计 235）

勾选后：
✓ 显示 42 个进程（总计 235）
```

**适用场景**：
- ✅ 快速找到占用内存的用户程序
- ✅ 避免误操作系统进程
- ✅ 聚焦于可以管理的进程

---

### 3. 智能排序

**排序规则**：
1. **优先级**：用户进程排在前面
2. **次要排序**：按已提交内存降序

**效果**：
```
排序前（只按内存）：
1. svchost      (系统) - 500 MB
2. chrome       (用户) - 450 MB
3. explorer     (系统) - 200 MB
4. notepad      (用户) - 150 MB

排序后（用户优先）：
1. chrome       (用户) - 450 MB  ← 用户进程区
2. notepad      (用户) - 150 MB
3. svchost      (系统) - 500 MB  ← 系统进程区
4. explorer     (系统) - 200 MB
```

**好处**：
- ✅ 用户进程更容易找到
- ✅ 高内存占用的用户程序一目了然
- ✅ 系统进程自动沉底

## 🎯 使用场景

### 场景 1：寻找高内存占用的用户程序

**步骤**：
1. 打开 MemCommitMonitor
2. 勾选"隐藏系统进程"
3. 查看列表顶部（已按内存降序）

**效果**：
- 快速找到 chrome、VS Code 等高内存程序
- 系统进程不干扰视线

---

### 场景 2：了解系统进程内存占用

**步骤**：
1. 打开 MemCommitMonitor
2. 不勾选"隐藏系统进程"
3. 滚动到列表底部

**效果**：
- 看到所有系统进程
- 灰色背景明显区分
- 无法误操作（右键菜单会拒绝）

---

### 场景 3：对比系统和用户内存

**步骤**：
1. 查看完整列表
2. 观察灰色区域（系统）和白色区域（用户）的占比

**效果**：
- 直观了解内存分配
- 决定应该管理哪些进程

## 🔍 技术细节

### 系统进程识别标准

**基于 ProcessAnalyzer 中的保护列表**：
```csharp
protected HashSet<string> ProtectedProcesses = {
    "System", "Registry", "smss", "csrss", "wininit",
    "services", "lsass", "winlogon", "dwm", "explorer",
    "svchost", "audiodg", "taskmgr", "fontdrvhost"
};
```

**标记逻辑**：
- IsProtected = true → 系统进程（灰色）
- IsProtected = false → 用户进程（白色）

### 颜色选择

**系统进程背景** (#E0E0E0)：
- 灰色但不太暗
- 明显区分但不刺眼
- 文字清晰可读

**系统进程文字** (#666666)：
- 中等灰度
- 与背景有足够对比度
- 暗示"不可操作"

## ⚙️ 配置建议

**日常使用**：
- ☑ 勾选"隐藏系统进程"
- 只关注用户程序
- 快速找到内存大户

**系统分析**：
- ☐ 不勾选"隐藏系统进程"
- 查看完整内存分布
- 了解系统开销

**内存清理**：
- ☑ 勾选"隐藏系统进程"
- 从高到低逐个检查
- 右键 → 查看详情 → 决定操作
