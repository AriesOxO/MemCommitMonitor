### 元素说明

**1. 背景 - 圆角方形**
- 尺寸: 256x256px

- 样式: 粗体、现代
- 位置: 居中偏上
- 含义:
  - **M**emCommit
  - **M**emory
  - **M**onitor

**3. 内存条图案**
- 位置: 底部装饰
- 数量: 3 条矩形
- 颜色: 白色半透明
  - 左: 50% opacity
  - 中: 38% opacity
  - 右: 100% opacity
- 含义: 模拟内存条的形态

## 🎯 设计原则

### macOS 风格

1. **扁平化** - 无阴影、无渐变（除背景）
2. **圆角** - 大圆角（48px）更友好
3. **简洁** - 去除多余元素
4. **几何** - 清晰的几何形状
5. **对比** - 高对比度的白色 M

### 可扩展性

- ✅ 矢量格式（XAML）可无限缩放
- ✅ 在小尺寸下依然清晰
- ✅ 黑白模式下可识别
- ✅ 适合各种背景色

## 🔧 技术实现

### XAML 矢量图形

**优点**：
- 无损缩放
- 文件体积小
- 易于修改颜色
- 支持动画（未来）

**实现方式**：
```xml
<DrawingImage x:Key="AppLogo">
    <DrawingImage.Drawing>
        <DrawingGroup>
            <!-- 背景 + M + 内存条 -->
        </DrawingGroup>
    </DrawingImage.Drawing>
</DrawingImage>
```

### 文件位置

```
MemCommitMonitor/
├─ Resources/
│  └─ Logo.xaml          ← Logo 定义
├─ MainWindow.xaml       ← 使用 Logo
└─ LogoPreview.xaml      ← Logo 预览工具
```

## 📝 使用指南

### 如何在代码中使用

**1. 引入资源**
```xml
<Window.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="Resources/Logo.xaml"/>
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Window.Resources>
```

**2. 设置窗口图标**
```xml
<Window Icon="{StaticResource AppLogoSmall}" ... >
```

**3. 在界面中使用**
```xml
<Image Source="{StaticResource AppLogo}"
       Width="128" Height="128"/>
```

### 如何预览 Logo

运行 `LogoPreview` 窗口查看不同尺寸的效果。

## 💡 设计思路

### 为什么是 M？

1. **品牌识别** - M 是 MemCommit Monitor 的首字母
2. **简洁明了** - 单个字母更容易记忆
3. **国际化** - 字母是通用符号

### 为什么加内存条图案？

1. **功能暗示** - 直观表达"内存"概念
2. **视觉平衡** - 填充底部空间
3. **细节丰富** - 增加设计层次

### 为什么用渐变？

1. **现代感** - 渐变是现代 UI 趋势
2. **立体感** - 增加视觉深度
3. **品质感** - macOS 风格常用渐变
