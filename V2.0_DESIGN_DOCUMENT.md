# v2.0 设计文档 - macOS 质量标准 UI 重构

**发布日期**: 2026-07-08  
**版本**: v2.0.1  
**设计理念**: 参考 macOS 的质量标准，打造优雅精致的现代界面

---

## 🎨 设计理念

### macOS 的设计哲学

- ✨ **简洁优雅** - 去除多余元素，突出核心内容
- 💎 **精致细节** - 每个像素都经过精心设计
- 🌊 **柔和舒适** - 柔和的阴影和圆角，减少视觉疲劳
- 🎯 **统一语言** - 所有元素遵循统一的设计系统
- 🎭 **细腻反馈** - 细腻的交互动画和状态变化

---

## 🎨 配色系统

### macOS 标准色板

```
背景色:
- MacOS.Background    #F5F5F7  (浅灰背景)
- MacOS.Card          #FFFFFF  (纯白卡片)
- MacOS.LightGray     #F2F2F7  (系统进程背景)

主题色:
- MacOS.Blue          #007AFF  (主色调)
- MacOS.BlueHover     #0051D5  (悬停状态)

语义色:
- MacOS.Red           #FF3B30  (危险操作)
- MacOS.RedHover      #D5291F  (红色悬停)
- MacOS.Orange        #FF9500  (警告)
- MacOS.OrangeHover   #D57700  (橙色悬停)
- MacOS.Green         #34C759  (成功状态)

文本色:
- MacOS.Text          #000000  (主文本)
- MacOS.TextSecondary #6C6C70  (辅助文本)
- MacOS.Gray          #8E8E93  (禁用/提示)

分隔线:
- MacOS.Divider       #E5E5EA  (边框/分隔)
```

### 颜色使用规范

**主色调（蓝色）**：
- 主要按钮背景
- 选中状态
- 链接文字
- 强调元素

**危险色（红色）**：
- 删除/终止按钮
- 错误提示
- 系统进程保护标记

**警告色（橙色）**：
- 实验性功能
- 警告提示
- 建议关闭标记

**成功色（绿色）**：
- 成功消息
- 状态指示

## 🧩 组件设计

### 1. 主窗口

**尺寸**：
- 默认: 1200×720
- 最小: 1000×600

**背景**: MacOS.Background (#F5F5F7)

**布局**：
- 外边距: 24px
- 区块间距: 20px

### 2. 卡片 (Card)

**样式**：
```xml
Background: White
CornerRadius: 10px
Padding: 20px
Shadow: 0 2px 12px rgba(0,0,0,0.06)
```

**使用场景**：
- 系统内存状态面板
- 进程列表容器

### 3. 按钮 (Button)

**主按钮**：
```xml
Background: MacOS.Blue
Foreground: White
FontSize: 13px
FontWeight: Medium
Padding: 16px 8px
Margin: 6px 0px
CornerRadius: 8px
Shadow: 0 2px 8px rgba(0,0,0,0.08)

Hover:
  Background: MacOS.BlueHover

Disabled:
  Opacity: 0.5
```

### 4. 复选框 (CheckBox)

**样式**：
```xml
FontSize: 13px
Foreground: MacOS.Text
VerticalAlignment: Center
```

### 5. 数据表格 (DataGrid)

**表格样式**：
```xml
GridLines: None
Background: Transparent
RowHeight: 44px
FontSize: 13px
```

**表头**：
```xml
Background: Transparent
Foreground: MacOS.TextSecondary
FontSize: 11px
FontWeight: Medium
Padding: 16px 12px
BorderBottom: 1px MacOS.Divider
```

**单元格**：
```xml

```

**行样式**：
```xml

  Background: White
  BorderBottom: 1px MacOS.Divider

  Foreground: MacOS.TextSecondary

  Background: #F0F0F5

  Background: #E3F2FD
```

### 6. 右键菜单 (ContextMenu)

**容器**：
```xml

CornerRadius: 8px
Padding: 4px

```

**菜单项**：
```xml

CornerRadius: 4px

Hover:
  Background: #E3F2FD

Disabled:
  Opacity: 0.5
```

**分隔线**：
```xml

Margin: 8px 6px
```

## ✨ 细节优化

### 1. 系统内存面板

**改进**：
- 双列布局（已提交 | 物理内存）
- 标签使用 Caption 样式（11px 灰色）
- 数值使用 Body 样式（13px Medium）
- 最后更新时间下沉到底部

### 2. 工具栏

**改进**：
- 左右分栏布局
- 左侧：功能按钮
- 右侧：状态 + 提示
- 统一的垂直对齐

### 3. 进程列表

**改进**：
- 无网格线（更简洁）
- 细分隔线（1px）
- 更大行高（44px）
- 更宽松的单元格内边距（16px）
- 表头与内容明确区分

### 4. 视觉层次

**强**：
- 标题文字（18px SemiBold）
- 选中行（浅蓝背景）
- 主要按钮（蓝色）

**中**：
- 正文文字（13px）
- 数值显示（13px Medium）
- 悬停状态

**弱**：
- 辅助文字（11px 灰色）
- 表头（11px 灰色）
- 分隔线（细线）
